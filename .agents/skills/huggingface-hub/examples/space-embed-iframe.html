Hugging Face Space Embedding Examples

Examples of embedding Hugging Face Spaces in your website.

1. Basic IFrame

Simple fixed-size iframe embedding.

<iframe
    src="https://nimaboscarino-hotdog-gradio.hf.space"
    frameborder="0"
    width="100%"
    height="450"
></iframe>

2. Responsive IFrame (16:9)

Maintains aspect ratio on all screen sizes.

<style>
.iframe-wrapper {
    position: relative;
    padding-bottom: 56.25%; /* 16:9 aspect ratio */
    height: 0;
    overflow: hidden;
}
.iframe-wrapper iframe {
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    border: none;
}
</style>

<div class="iframe-wrapper">
    <iframe src="https://nimaboscarino-hotdog-gradio.hf.space"></iframe>
</div>

3. Gradio WebComponent

Better performance and auto-sizing (Gradio Spaces only).

<!-- Load Gradio JS (match version in Space) -->
<script
    type="module"
    src="https://gradio.s3-us-west-2.amazonaws.com/5.0.0/gradio.js"
></script>

<!-- Embed the Space -->
<gradio-app src="https://nimaboscarino-hotdog-gradio.hf.space"></gradio-app>
Note: WebComponents require the Gradio JS library to be loaded. Match the version to the Space's sdk_version.

4. Multiple Spaces Grid

Display multiple Spaces in a grid layout.

<style>
.spaces-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(400px, 1fr));
    gap: 20px;
}
.space-card {
    border: 1px solid #ddd;
    border-radius: 8px;
    overflow: hidden;
}
.space-card h4 {
    margin: 0;
    padding: 10px;
    background: #f0f0f0;
}
</style>

<div class="spaces-grid">
    <div class="space-card">
        <h4>Image Classifier</h4>
        <iframe src="https://space1.hf.space" height="300"></iframe>
    </div>
    <div class="space-card">
        <h4>Text Generator</h4>
        <iframe src="https://space2.hf.space" height="300"></iframe>
    </div>
</div>

5. Lazy Loading (Performance)

Only load Space when scrolled into view.

<iframe
    src="https://username-spacename.hf.space"
    loading="lazy"
    width="100%"
    height="450"
></iframe>
Tip: Use loading="lazy" for Spaces below the fold to improve page load performance.

6. With Loading Placeholder

Show a placeholder while Space loads.

<style>
.space-container {
    position: relative;
    width: 100%;
    height: 450px;
    background: #f0f0f0;
}
.space-container::before {
    content: "Loading Space...";
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    color: #666;
}
.space-container iframe {
    position: relative;
    z-index: 1;
    width: 100%;
    height: 100%;
    border: none;
}
</style>

<div class="space-container">
    <iframe src="https://username-spacename.hf.space"></iframe>
</div>

7. Full Page Embed

Make Space fill the entire page.

<!DOCTYPE html>
<html>
<head>
    <style>
        html, body {
            margin: 0;
            padding: 0;
            height: 100%;
            overflow: hidden;
        }
        iframe {
            width: 100%;
            height: 100%;
            border: none;
        }
    </style>
</head>
<body>
    <iframe src="https://username-spacename.hf.space"></iframe>
</body>
</html>

8. React Component

Reusable React component for embedding Spaces.

// HFSpace.jsx
import React from 'react';

export function HFSpace({ username, spaceName, height = 450 }) {
    const url = `https://${username}-${spaceName}.hf.space`;

    return (
        <iframe
            src={url}
            style={{
                width: '100%',
                height: `${height}px`,
                border: 'none',
                borderRadius: '8px'
            }}
            loading="lazy"
            title={`${username}/${spaceName} Space`}
        />
    );
}

// Usage:
// <HFSpace username="gradio" spaceName="text-generation" height={500} />

Tips & Best Practices

  • HTTPS required: Spaces use HTTPS, so your page must also use HTTPS
  • Cross-origin: Spaces run in a different origin, so JavaScript communication is limited
  • Mobile: Test on mobile devices - some Spaces may need more height
  • Loading: Use loading="lazy" for better performance
  • Accessibility: Add meaningful title attribute to iframes
  • Private Spaces: Cannot be embedded - viewers need to be logged in
